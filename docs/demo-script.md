# OpsBot Demo Script

## Demo Flows

1. **Check-in**
   User taps "Check-in".
   OpsBot says: "Welcome. Check-in closes at 10:30 and hacking starts at 10:30."
   Robot action: `point_checkin`

2. **Lost item**
   User taps "Lost item".
   OpsBot says: "Please place the item in Lost & Found. I'll register it for staff."
   Robot action: `point_lost_found`

3. **Charger request**
   User taps "Charger request".
   OpsBot says: "You can ask staff for a charger or check the shared accessories table."
   Robot action: `point_charger`

4. **Demo schedule**
   User taps "Demo schedule".
   OpsBot says: "Code freeze is at 17:00 and live demos start at 17:30."
   Robot action: `point_demo_queue`
