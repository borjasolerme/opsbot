# OpsBot Supabase

Supabase backend layer for Edge Functions, Postgres logs, and later integration events.

## Scope

- Edge Function endpoint for visitor intent handling.
- Postgres tables for action and interaction logs.
- Service-side integration point for ScrapeGraph, Interhuman, and Cyberwave.

## Current Status

No Supabase functionality is implemented yet. This folder only reserves the backend structure.
